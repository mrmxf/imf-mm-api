imf-mm-api Home

This is the the demo API server and reference code. Documentation is on github.

The crawl functionality will crawl the IMF resouces in the folders specified in config.json. By default these are the small test files supplied with the demo software

When the database is populated with resources you can fire queries from the demo page to test everything is working.

If you're going to develop the software then look at the supplied README.md and be sure to run the test suite before hacking away.

If you're just going to use the software to test out no-file-name workflows then update config.json to point at a folder of IMF assets and off you go.
